# 🎬 AnimaSérie

Site de catálogo de filmes, séries e animes com sistema de avaliação por estrelas. Projeto desenvolvido para a disciplina de Programação Web I.

## 📸 Preview

![Preview do site](https://lauraacioly.github.io/Projeto-Prog-Web/)

🔗 **[Acesse o site ao vivo](https://lauraacioly.github.io/Projeto-Prog-Web/)**

## 📌 Funcionalidades

- Página inicial com catálogo organizado por categorias (Filmes, Séries e Animes)
- Navegação entre páginas por menu
- Páginas individuais com sinopse, avaliação por estrelas e onde assistir
- Campo de pesquisa por título
- Layout responsivo com tema visual personalizado

## 📁 Estrutura do projeto

```
Projeto-Prog-Web/
├── index.html          # Página principal com catálogo
├── style.css           # Estilos globais
├── imagens/            # Imagens do site
├── paginaum/           # Página de detalhe 1
├── paginadois/         # Página de detalhe 2
├── paginatres/         # Página de detalhe 3
└── paginaquatro/       # Página de detalhe 4
```

## 💡 Aprendizados

- Estruturação de site multipágina com HTML
- Estilização com CSS (layout, cores, tipografia)
- Organização de arquivos e pastas em um projeto web
- Navegação entre páginas com links internos
